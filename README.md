# Bank-Data-Analytics
# Project Over view
This project focuses on analyzing banking data to gain insights into customer behavior, financial performance, and operational efficiency. The analysis helps banks make informed decisions related to customer segmentation, loan management, and risk assessment.

# Objectives
Analyze customer and transaction data to identify patterns.

Understand factors affecting loans and customer behavior.

Support data-driven decision-making in the banking sector.

# Key Analysis Performed
Customer segmentation analysis.

Loan approval and default analysis.

Transaction and balance trend analysis.

Risk and performance analysis.

# Tools
Microsoft Excel
Power BI 
Tableau
SQL 


KPI dashboards for banking insights
